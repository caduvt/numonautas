# Numonautas

Numonautas é um jogo que utiliza sistemas digitais para auxiliar o ensino de matemática para crianças dentro do espectro autista. Por meio de uma placa FPGA conectada a um notebook, são exibidas uma série de expressões matemáticas com dificuldade crescente. O objetivo da criança é responder corretamente às questões, aprendeendo desde operações elementares, como soma e subtração, até radiciação e potenciação. 

O jogo visa um público alvo de crianças com TEA nível 1, entre 5 e 13 anos de idade.
